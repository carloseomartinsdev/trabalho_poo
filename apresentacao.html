🏢 1. Contexto


Problema: Uma empresa de logística em Fortaleza/CE precisa realizar entregas diárias em 15 clientes, partindo de um centro de distribuição, usando 2 veículos com capacidade limitada. As rotas são definidas manualmente, sem critério de otimização.

Pergunta-chave: Compensa adicionar um 3º veículo (+ R$ 3.000/mês) se a distância total diminuir?

📐 2. Modelagem — CVRP


Técnica: Programação Inteira (PI)
Tipo: CVRP — Capacitated Vehicle Routing Problem
Distâncias: Calculadas via API OSRM (rotas rodoviárias reais)

ElementoDescrição
Variável de decisãox[i][j][k] = 1 se veículo k vai do ponto i ao ponto j
Função objetivoMinimizar Σ distância(i,j) × x[i][j][k]
FerramentaPython + PuLP (solver CBC) + OSRM API

🔒 Restrições do Modelo


  1. Cada cliente é visitado exatamente 1 vez
  2. Cada veículo sai e retorna ao CD
  3. Conservação de fluxo — se entra, sai
  4. Capacidade máxima de 500kg por veículo
  5. Eliminação de sub-rotas (Miller-Tucker-Zemlin)

MTZ: u[i] − u[j] + N × x[i][j][k] ≤ N − 1    ∀ i,j ∈ clientes, i ≠ j

💡 Insights Descobertos


🔍 Decisão subótima: A rota manual gera 44.2% mais distância. A otimização matemática comprova economia de R$ 140 mil/ano.
⚖️ Trade-off revelado: Adicionar um 3º veículo reduz apenas 1.4% da distância, mas custa R$ 3.000/mês a mais — não compensa.
🔧 Gargalo escondido: A capacidade dos veículos (494/500 e 479/500 kg) é a restrição que mais impacta a distribuição dos clientes.
