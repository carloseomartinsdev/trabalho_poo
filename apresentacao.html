2. Metodologia


Como transformamos o problema em um modelo matemático?

Técnica: Programação Inteira (PI)
Modelo: CVRP — Capacitated Vehicle Routing Problem
Distâncias: Calculadas via API OSRM (rotas rodoviárias reais de Fortaleza)

ElementoDescrição
Variável de decisãox[i][j][k] = 1 se veículo k vai do ponto i ao ponto j
Função objetivoMinimizar Σ distância(i,j) × x[i][j][k]
FerramentaPython + PuLP (solver CBC) + OSRM API

Quais regras o modelo precisa respeitar?


  1. Cada cliente é visitado exatamente 1 vez
  2. Cada veículo sai e retorna ao centro de distribuição
  3. Conservação de fluxo — se um veículo entra, ele precisa sair
  4. Capacidade máxima de 500kg por veículo
  5. Eliminação de sub-rotas (Miller-Tucker-Zemlin)

MTZ: u[i] − u[j] + N × x[i][j][k] ≤ N − 1    ∀ i,j ∈ clientes, i ≠ j

Por que MTZ? Sem essa restrição, o solver poderia criar "mini-rotas" desconectadas do depósito. A MTZ garante que todas as rotas passem pelo CD.

Se a demanda crescer 28 kg, o 3º veículo se torna obrigatório.

💡 Insight estratégico: O 3º veículo não se justifica por economia de km (prejuízo de R$ 2.797/mês), mas pode se tornar necessário por viabilidade operacional se a demanda ultrapassar 1.000 kg. A empresa deve monitorar a demanda diária.

4. Conclusão


O que aprendemos com este projeto?


🔍 Decisão subótima: Mesmo com motorista experiente (vizinho mais próximo), a rota manual gera 17.1% mais distância. A otimização gera economia de R$ 36 mil/ano só em km.
⏱️ Custo oculto: Considerando o tempo no trânsito de Fortaleza, a economia sobe para ~R$ 52 mil/ano (~45% a mais).
⚖️ Trade-off: Adicionar um 3º veículo reduz apenas 1.4% da distância, mas custa R$ 3.000/mês a mais. Não compensa.
🔧 Gargalo: A capacidade dos veículos (~98% ocupada) é o fator que mais limita a eficiência. Com 1 veículo, a operação é inviável.

Quais são as limitações e o que pode ser melhorado?


Limitações

  • Distâncias OSRM — sem trânsito em tempo real
  • Sem janelas de horário para entregas
  • Custo por km fixo (sem variações)
  • Velocidade média estimada (não medida)
